# AI Access Policy

**Effective Date:** This Policy applies from the date of publication and remains in effect until amended or replaced.
Last updated: 27th of December, 2025

This AI Access Policy (“Policy”) governs all access to this website (“the Website”) by automated systems, agents, or processes used for artificial intelligence, machine learning, dataset creation, or related computational purposes. This Policy is legally binding and forms part of the conditions under which access to the Website is permitted.

## 1. Purpose and Scope

1.1. This Policy applies exclusively to automated systems, bots, crawlers, agents, or scripts that access the Website for purposes related to:
- Artificial intelligence
- Machine learning
- Model training
- Dataset creation
- Fine‑tuning
- Embedding generation
- Algorithmic analysis
- Any computational process intended to develop, improve, or operate AI systems.

1.2. This Policy does not apply to ordinary non‑AI crawlers, search engine indexing bots, uptime monitors, or automated systems that are not engaged in AI‑related data collection or model development.

## 2. Prohibition on AI‑Related Automated Access

2.1. AI‑related automated access to the Website is strictly prohibited unless the operator has obtained explicit prior written permission from the Website Owner.

2.2. Prohibited AI‑related access includes, without limitation:
- Scraping or harvesting text, images, metadata, or structural information
- Collecting data for training, fine‑tuning, or evaluating AI models
- Generating embeddings or datasets
- Using the Website as input to any machine learning pipeline

2.3. Absence of written permission constitutes lack of authorization.

## 3. Licensing Requirement

3.1. Any party seeking to use the Website’s content, structure, or data for AI‑related purposes must enter into a written licensing agreement with the Website Owner prior to accessing the Website.

3.2. Permission is not granted by default and must be explicitly confirmed in writing.

3.3. The Website Owner may approve, deny, or revoke permission at its sole discretion.

4.2. The operator of the AI system is responsible for all fees, regardless of whether the access was intentional, unintentional, or performed by a third‑party agent acting on their behalf.

5.1. By accessing the Website for AI‑related purposes, automated agents and their operators agree to be bound by this Policy, regardless of whether the operator has reviewed it. This Policy is linked clearly in the Website's robots.txt file.
